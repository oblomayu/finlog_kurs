# FinLog Web

Веб-приложение для управления личными финансами с автоматической категоризацией расходов на основе правил.  
Создано на базе консольного приложения FinLog в рамках курсовой работы.

## Возможности
- Добавление, просмотр, редактирование, удаление транзакций (доход/расход)
- Фильтрация по типу, категории, дате
- Управление месячными бюджетами по категориям с прогресс-барами
- **Автоматическая категоризация расходов** по правилам (regex или подстрока)
- Настройка правил: приоритет, активность, поле для проверки
- Экспорт всех данных в ZIP-архив (CSV + SQLite + JSON бюджетов)
- Импорт из ранее экспортированного архива
- Автоматическое резервное копирование базы при запуске
- Логирование событий в `app.log`
- Настройка через `.env`

## Технологии
- Python 3.10+, Flask 3.x
- SQLite, Jinja2, Bootstrap 5
- python-dotenv, logging

## Установка и запуск

1. Установка виртуального окружения:
- python -m venv venv
- source venv/bin/activate   # Linux/macOS
- venv\Scripts\activate      # Windows
- pip install -r requirements.txt
2. Запуск проекта:
- python3 run.py
3. Веб-приложение будет доступно в браузере по ссылке: http://127.0.0.1:5000 или http://localhost:5000
